# Электронный дневник (Electronic diary)

Веб-приложение для учителей: управление учениками, выставление оценок, статистика успеваемости.  
Разработано в качестве пет-проекта для демонстрации навыков работы с Python, Django, Bootstrap, Docker.

## Функциональность

-  Регистрация и аутентификация учителей и администратора
-  Панель учителя с метриками (последние оценки, средний балл, двоечники)
-  CRUD для учеников и оценок
-  Детальная страница ученика с историей оценок по предметам
-  Статистика системы (общая успеваемость, топ предметов, активность)
-  Фильтрация и поиск учеников по классу и ФИО
-  Адаптивный интерфейс на Bootstrap 5
-  Готовый Docker-контейнер для быстрого запуска

## Технологии

- **Backend:** Python 3.12, Django 5.0.2, SQLite (включена в репозиторий с демо-данными)
- **Frontend:** Bootstrap 5, Bootstrap Icons, HTML-шаблоны
- **Дополнительно:** django-bootstrap5, django-crispy-forms, django-filter, django-tables2
- **Инструменты:** Gunicorn, Whitenoise, Docker, Docker Compose

## Быстрый старт

### Запуск через Docker (рекомендуемый способ)

1. Убедитесь, что на вашем компьютере установлены Docker и Docker Compose

2. Клонируйте репозиторий

3. Перейдите в корневую папку и запустите контейнер
cd electronic_diary
docker-compose up

4. Дождитесь следующих сообщений, откройте браузер и перейдите по указанному адресу
Сервер доступен по адресу http://localhost:8000
Для остановки сервера нажмите Ctrl+C

5. База данных уже содержит демо-учеников, предметы и оценки. При первом запуске миграции применятся автоматически.

### Локальный запуск (без Docker)

1. Убедитесь, что установлен Python 3.12 и клонируйте репозиторий

2. Перейдите в папку проекта, создайте и активируйте виртуальное окружение
cd electronic_diary
python -m venv .venv
.venv\Scripts\activate   

3. Установите зависимости и примените миграции
pip install -r requirements.txt
python manage.py migrate

4. Запустите сервер и перейдите по указанному адресу (http://127.0.0.1:8000/)
python manage.py runserver

### Создание суперпользователя (для доступа в админку)

1. Если вы хотите войти в стандартную административную панель Django, выполните
docker-compose exec web python manage.py createsuperuser   # для Docker
python manage.py createsuperuser                           # для локального запуска

2. Затем перейдите по адресу /admin/ и используйте созданные учётные данные
